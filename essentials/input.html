input type="checkbox" 当 v-model 绑定的值为 boolean 时，自动切换。
input 事件总是先于 change 事件被调用。
input 事件中会改变 v-model 绑定的 boolean 值，而 change 不会。因此如果需要使用这个 boolean 值进行计算，要写在 change 事件中。
多选/全选操作：点击选中时，绑定的值会被加入到 v-model (一个一个选)；反之当 v-model 中有了该值，会被自动选中 (全选)。
